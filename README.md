# Ride in Hand
## Introduction
This is a **Ride Sharing Application** that helps users to interact with eachother and share rides. The users ride holders can earn money and the riders can travel easily by the sharing mechanism.
## Technologies Used
* Android SDK
* Google Firebase
  * Authentication
  * Cloud Firestore
  * Storage
  * Cloud Functions
* Google Maps SDK
  * Geocoding
  * Directions API
  * Places API
